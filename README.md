## Layout

- `outlines/` — the book outline
- `chapter_notes/`, `chapter_outlines/` — co-produced notes+outline pairs per chapter/section
- `chapter_drafts/chapter-NN/` — manuscript prose + `figures/`
- `agreements/` — writing process and style rules (self-contained copy)
- `proposals/` — the Springer proposal
